基本概念

齐次坐标的思路是增加一个左边：

  • 对于2D point，利用$(x, y,1)^T$表示；
  • 对于2D vector，利用$(x,y,0)^T$表示；

那么平移变换可以在齐次坐标下表示为：

讨论

在齐次坐标下，有如下等式成立：

  • vector + vector = vector；
  • point - point = vector；
  • point + vector = point；
  • point + point = 两点的中点；
    • 利用性质$(x, y, w)=(x/w, y/w, 1)$；
