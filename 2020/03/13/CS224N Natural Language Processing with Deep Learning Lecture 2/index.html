Dealing With Ambiguity（处理歧义）

由于很多词有一词多义的情况，老师介绍了一种处理该问题的方法：

  1. 选择上下文窗口长度（例如$5$），计算Co-occurrence Matrix
  2. 每个上下文用上下文单词向量的加权和表示（idf-weighting）
  3. 使用k-means对上下文向量进行聚类
  4. 最终，每个单词都被重新标记到其关联的簇，并用于训练该簇的单词表示。
