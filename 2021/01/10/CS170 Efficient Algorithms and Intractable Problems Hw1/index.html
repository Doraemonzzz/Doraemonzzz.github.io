Main idea:

因为是有序的数组，所以采用二分法。

l = 0
r = n - 1
while (l < r):
	m = (l + r) / 2
	if A[m] < k:
        l = m + 1
     else:
        r = m

if (A[l] != k):
    return -1
else:
	return l

Proof of correctness:

如果A[m] < k，那么位置必然大于m；如果A[m] >= k，那么位置必然小于等于m。
