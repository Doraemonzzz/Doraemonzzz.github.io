3. MST Variant

这部分参考了习题解答。

伪代码如下：

  • 在$G=(V,E)$上计算MST，得到$T_0$，权重为$w(T_0)$
  • 初始化$T_1= T_0$
  • for $s\in S$:
    • $T_2=T_0\cup \{s\}$
    • 在$T_2$包含的环中删除权重最大边。
    • if $w(T_1) < w(T_2)$:
      • $T_1 = T_2$
  • return $T_1 $

算法是简洁明了的，下面证明其正确性。

如果$T_2$不包含属于$S$的边，那么$T_1=T_2= T_0$，结论成立。
