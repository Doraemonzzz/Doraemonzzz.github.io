Flarum插件安装（一）

发表于2021-07-04|更新于2021-07-04|论坛|Flarum
|字数总计:1.3k|阅读时长:5分钟|阅读量:|评论数:

这部分给出论坛已经安装的插件以及对应功能，方便后续更新，查找问题。

准备工作

如果安装时报如下错误：

As there is no 'unzip' nor '7z' command installed zip files are being unpacked using the PHP zip extension.

那么运行如下命令即可：

sudo apt-get install unzip

启用插件

中文插件

链接：

  • https://extiverse.com/extension/flarum-lang/chinese-simplified

数学公式

链接：

  • https://extiverse.com/extension/if/flajax

说明：

基本功能测试无误，角标需要转义，例如

a_, a^, \\

应该转换为

a\_, a\^, \\\

相关资料：

  • https://discuss.flarum.org/d/27457-mathjax-plugin-for-flarum

第三方登录（非国内账号）

链接：

  • https://extiverse.com/extension/fof/oauth

配置教程：

  • https://discuss.flarum.org.cn/d/246

说明：

github, google测试通过，关联的邮箱为社交账号对应的邮箱，论坛的唯一标识是邮箱。

配置教程：

  • https://extiverse.com/extension/extiverse/mercury

相关资料：

  • https://discuss.flarum.org/d/27620-mercury-the-extiverse-extension-to-understand-extension-updates

  • 后台参数在setting表中，如果后台挂了，可以直接改setting表中对应内容。

  • 如果后台卡住，调用如下命令即可：

    php flarum mercury:update-check

display: none!important;

     .item-forumStatisticsWidget {
         display: none;
    }
}
